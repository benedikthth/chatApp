# ChatApp
## Instructions to Run:
1. $ git clone -> repository!
2.
'''bash
 $ npm install
 $ bower install
 $ gulp build
'''

6.
'''bash
 $ node server/server.js
'''
7. Run http server from the folder that contains index.html
8. Access The project via http://localhost:8000
## Server Setup
'''bash
$ cd server/
$ npm install
$ node chatserver.js
'''
